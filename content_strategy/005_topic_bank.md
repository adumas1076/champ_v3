- What funnel slot needs filling today
- Which pillar hasn't been touched this week
- Which topics are adjacent to trending external events
- Which hook types underperformed last week (rotation)

**Daily Content Pull:**
```
Morning TOFU slot → pull topic from TOFU pool
  → Match to pillar rotation (today = Pillar 3)
  → Match to bucket (Contrarian this morning)
  → Match to platform slot (Twitter)
  → Load hook from 004_hook_library
  → Pass to Step 4 (Thread Check) for series connection
  → Continue through Content Factory
```

**Topic Depletion + Refresh:**
- After a topic is produced and published, mark as USED
- Weekly autoresearch loop scans for new topics based on:
  - Outlier research (what's winning in watch-list creators' niches)
  - External events (news, trends, industry drops)
  - Performance patterns (if Mindset topics are hitting, generate 10 more)
