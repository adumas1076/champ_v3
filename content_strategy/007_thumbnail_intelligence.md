## UNIVERSAL THUMBNAIL PRINCIPLES (Cross-Platform)

These rules apply regardless of platform:

### 1. The 3-Second Rule
If someone can't grasp what the content is about in 3 seconds of looking at the thumbnail, it fails. Test: show thumbnail to a stranger for 3 seconds, ask them what it's about. If they hesitate, simplify.

### 2. The Thumb-Test
On mobile, thumbnails compete with 50+ others in a feed. Can yours be read at thumbnail size (the actual tiny preview)? If text is unreadable small, make the text bigger.

### 3. The Emotion Trigger
Face + expression = highest CTR across all platforms. Neutral face = dead. Surprised, laughing, intense, or "what the hell" expressions = clickable.

### 4. The Contrast Rule
Dark bg + light text, OR light bg + dark text. Never same-tone on same-tone. Creates visual pop on a scrolling feed.

### 5. The One-Idea Rule
Thumbnail should communicate ONE idea, ONE promise, ONE curiosity gap. Multiple messages = muddy = skipped.

## PLATFORM-SPECIFIC THUMBNAIL RULES

### YouTube (Long-Form Pillar)

**Winning pattern from 2026 data:**
- Close-up face + bold text overlay + high-contrast background
- Face occupies 30-40% of frame
- Text: 3-4 words MAX
- Text placement: left or right third, never center (center gets covered by duration stamp)
- Colors: high contrast, brand accent color dominant

**Outlier study — what top performers do:**
| Top YT Creator Style | What to Steal |
|----------------------|---------------|
| Alex Hormozi | Big face + yellow/white bold text + minimal bg clutter |
| MrBeast-style | Shocked face + circled object + bright contrast |
| Dan Koe | Aesthetic bg + clean typography + face optional |
| VidIQ/TubeBuddy patterns | Outlined text, high saturation, face eye contact |

**Anthony's YouTube template:**
```
Frame: Anthony center-right, dark bg with yellow brush accent behind
Face: Looking at camera, slight intensity (not smiling neutral)
Text: Left third, 3-4 words in Montage font, yellow or white
Sub-text: Below main text, smaller, white
Brand mark: Bottom right corner, "Creator Create" tag
```

### Instagram Reels / TikTok / Shorts

**Winning pattern from 2026 data:**
- Cover frame is critical (shows in Explore + feed + profile grid)
- Text-first or face-first (never both competing)
- Vertical composition (1080x1920)
- Bold cover text = same as hook of the video

**Outlier pattern:**
- Top 1% of Reels use text overlays that summarize the ENTIRE video's promise
- "That's true but that's not the truth" style covers win because the catchphrase IS the hook
- Face + text STACKED (text top half, face bottom half) outperforms side-by-side

**Anthony's Reels cover template:**
```
Background: Dark, with optional yellow brush accent PNG
Top 40%: Hook text (3-7 words, Montage font, yellow or white)
Middle 40%: Anthony's face or scene image
Bottom 20%: Subtitle text + "Creator Create" brand tag
Caption style: Highlighted word-by-word (matches content style)
```

### Instagram Feed (Square/Portrait)

**Winning pattern:**
- Aesthetic + text-driven (not photo-driven)
- Square format 1080x1080 or portrait 1080x1350
- Text is the hero, image is support
- Carousel covers especially need strong text (swipe bait)

**Anthony's feed template:**
```
Background: Dark or gradient (brand palette)
Headline: Center-top, large, bold
Accent: Yellow brush underline or background stroke
Footer: "Creator Create" or "Think it → Vibe it → Have it"
```

### Twitter/X

**Winning pattern:**
- Image supports tweet, doesn't repeat it
- Simple bold text over color block
- Memes/charts/screenshots > faces (counterintuitive — on X, info > persona)
- 1200x675 or 1080x1080

**Anthony's Twitter template:**
```
For text hooks: Quote card — dark bg + yellow brush accent + bold text
For stats/proof: Chart or screenshot with Anthony's markup overlay
For hot takes: Text-only card with contrarian statement
```

### LinkedIn

**Winning pattern:**
- Professional but not boring
- Text + brand colors dominant
- Carousel format crushes (10 slides wins over single image)
- Business-casual tone, not hype-y

**Anthony's LinkedIn template:**
```
Single post: Quote card or framework diagram
Carousel: Cover slide (hook), 8 value slides, CTA slide
Colors: Brand palette, darker variant (LinkedIn audience = business)
Text: Headline font, readable at thumbnail size
```

## THUMBNAIL A/B TESTING STRATEGY

### When to A/B Test
- YouTube pillar videos: ALWAYS (native A/B testing available)
- IG Reels / TikTok: Test 2 cover frames on medium-stakes content
- Twitter / LinkedIn: Test 2 images on high-stakes content
- Daily TOFU posts: Skip testing (speed > optimization)

### What to Test
**Variable 1: Face vs No-Face**
- Hypothesis: Face drives click but text-only dominates in certain niches
- Run both, let data decide per bucket

**Variable 2: Text Placement**
- Top vs Bottom vs Center
- Usually top wins on mobile (feed scroll direction)

**Variable 3: Color Accent**
- Yellow/gold vs Red vs White
- Red for contrarian, yellow for mindset/education, white for proof

**Variable 4: Expression**
- Neutral vs Intense vs Laughing vs "What the hell"
- Intense typically wins for Anthony's brand (matches contrarian energy)

### Pre-Score Formula (When Split Testing Isn't Available)
```
THUMBNAIL SCORE =
  Face present + clear expression     (1-10)
  + Text readability at small size    (1-10)
  + Contrast ratio                    (1-10)
  + Brand consistency                 (1-10)
  + Pattern match to top outliers     (1-10)
  / 5 = average score

Must be 7+ to ship.
```
